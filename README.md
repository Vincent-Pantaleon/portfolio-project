# Vincent Pantaleon — Portfolio

A personal portfolio website built with React, Vite, TypeScript, and Tailwind CSS.

## Tech Stack

- **Language:** TypeScript
- **Frontend:** React, Tailwind CSS, Vite
- **Email Handling:** Formspree.io

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Vincent-Pantaleon/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set up environment variables

Create a `.env` file in the root of the project:
- `VITE_GITHUB_USERNAME` — your GitHub username
- `VITE_EMAIL` — the email displayed in the contact section
- `VITE_FORMSPREE_ID` — your form ID from [formspree.io](https://formspree.io)

> ⚠️ Never commit your `.env` file. Make sure it's listed in `.gitignore`.

### 4. Run the development server

```bash
npm run dev
```

### 5. Build for production

```bash
npm run build
```
